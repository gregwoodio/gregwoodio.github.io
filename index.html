PuzzleScript

  • A Puzzling Snowman is Hard to Build (2021?) - Inspired by the game A Good Snowman is Hard to Build by Alan Hazelden & Benjamin Davis. It's a Sokoban/pushing block puzzle game where you need to stack three snowballs of increasing size to build a snowman. If you don't have the right size snowballs then you can push a smaller one through a patch of snow to make it bigger. PuzzleScript is very graphically limited which makes this a bit hard to play, but once you start to recognize each game piece it's actually a pretty fun game.
  • Colourmancer (2021?) - Yes there's a 'u' in it. Guide your wizard through the mazes by pushing keys to locks in order to raise and lower magical walls.
